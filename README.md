# TorrentPieceChecker
## 种子分块校验工具
[![Ask DeepWiki](https://deepwiki.com/badge.svg)](https://deepwiki.com/zhaoyangwx/TorrentPieceChecker)

按照指定分块进行校验，方便分批多次核对。校验过的分块自动取消勾选。
